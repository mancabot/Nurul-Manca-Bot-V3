## Install
Follow The Steps Below!

```bash
> termux-setup-storage
(after that tap on permission)
> pkg update -y
> pkg upgrade -y
> pkg install git -y
> git clone https://github.com/mancabot/Nurul-Manca-Bot-V3
> cd Nurul-Manca-Bot-V3
> npm cache clear
> bash install.sh
> npm audit fix
> npm start / node index.js
```
